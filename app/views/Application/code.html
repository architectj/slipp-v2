  • 라이센스

    소스 코드에 대한 라이센스는 아직까지 특별히 고민한 정책이 없다. 이 소스 코드에 적합하리라 생각하는 라이센스가 있으면 제안해 주면 좋겠다.

  • 기능제안 및 버그신고

    slipp.net을 사용하는 중 새로운 기능을 제공, 기존 기능에 대한 개선, 버그가 있다면 SLiPP github에 남길 수 있다.

  • 소스코드 다운로드

    slipp.net은 두 개의 프로젝트로 구성되어 있어 있다.
    www.slipp.net을 서비스하는 메인 프로젝트와 파일의 업로드를 담당하는 attaches.slipp.net이다.

    www.slipp.net (https://github.com/javajigi/slipp)

    git clone git@github.com:javajigi/slipp.git

    attaches.slipp.net (https://github.com/javajigi/slipp-uploader)

    git clone git@github.com:javajigi/slipp-uploader.git

    현재 서비스하고 있는 slipp.net은 google app engine + play framework로 구성되어 있다.
    이전 버전은 google app engine + spring framework + objectify framework로 구성되어 있다.
    이 소스는 다음 저장소에서 다운로드 받을 수 있다.

    slipp.net 최초 소스 (https://github.com/javajigi/slipp-spring)

    git clone git@github.com:javajigi/slipp-spring.git

  • 데모 및 테스트

    slipp.net은 개발과 운영을 분리해서 운영하고 있다.
    현재 개발되고 있는 slipp.net은 http://dev.slipp.net에서 확인할 수 있다.
    http://dev.slipp.net은 개발 및 테스트를 위한 목적이기 때문에 부담을 가지지 않고 마음껏 테스트를 해도 된다.
